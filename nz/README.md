Download.

1. Go to https://www.stats.govt.nz/experimental/covid-19-data-portal.
2. Click "Download data".
3. Select "Health - Weekly deaths - By region".
4. Select "Full range".

Format.

```
./main.R covid_19_data_portal.csv > nz-deaths.csv
```
